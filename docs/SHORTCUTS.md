# Keyboard Shortcuts

## General

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl+K` | Open Spotlight Search |
| `Cmd/Ctrl+Shift+M` | Toggle Pro Mode |
| `Cmd/Ctrl+Shift+F` | Toggle Flow Mode (in Pro Mode) |
| `Shift+?` | Show shortcuts overlay |
| `Space` | Play / Pause |
| `` ` `` (backtick) | Toggle shortcuts overlay |

## Pro Mode (ChromicEditor)

| Shortcut | Action |
|----------|--------|
| `[` / `]` | Shift word timing earlier / later |
| `Shift+Click` | Seek to lyric line |
| `Cmd+Enter` | Convert cue to line |
| `Cmd+S` | Export JSON |
| `Cmd+Shift+S` | Export LRC |
| `Cmd+Backspace` | Convert line to vocal cue |
| `Cmd+Shift+Backspace` | Delete element |

## Playback

| Shortcut | Action |
|----------|--------|
| `Space` | Play / Pause |
| `Left/Right` | Seek -5s / +5s |
| `Cmd+Left/Right` | Previous / Next track |

> **Note:** On Windows/Linux, replace `Cmd` with `Ctrl`.
